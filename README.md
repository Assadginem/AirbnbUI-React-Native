# AirbnbUI-React-Native


A Airbnb User Interface created in React Native

Some features and libraries include: 
   - React Navigation
   - React native star rating
   - Native Base
   - Horizontal Scrolling
   - Grid component examples
   
   1. Pull repo
   2. cd into repo
   3. yarn install
   4. yarn start 
   5. run on ios or android
